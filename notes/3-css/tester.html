Understanding box model

Box sizes can vary depending on the browser and its rules.

        .this-box {
            border: 1px solid blue;
            width: 300px;
            height: 300px;
            padding: 30px;
          }
THIS BOX

Though has height and width of 300px, overall size will be 662px x 662px due to padding added and border added


-----------

        .that-box {
            border: 1px solid blue;
            width: 300px;
            height: 300px;
            padding: 30px;
            box-sizing: border-box;
          }

"box-sizing: border-box;" tells preview to make overall size to the stated height and width of 300px x 300px

THAT BOX


-----------

        .that-box-too {
            border: 1px solid blue;
            width: 300px;
            height: 300px;
            padding: 30px;
            box-sizing: content-box;
            outline: 3px solid black;
            outline-offset: 10px;
          }

Note codes in bold above. Outline is not included in size and can also be stated to be offset.
